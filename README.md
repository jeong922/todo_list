## 화면 구성

- All : Active, Completed 상관없이 전체 리스트를 보여준다.
- Active : 체크박스로 체크 되어지지 않은 아이템의 리스트들만 보여준다.
- Completed : 체크박스로 체크 되어 있는 아이템의 리스트들만 보여준다.
- 왼쪽 상단 🌞 또는 🌙 모양 버튼을 누르면 다크모드 또는 라이트모드로 변경 된다.
